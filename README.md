# Docster


Transforming proxy that creates an html represenation of your json hypermedia (e.g. HAL) api on the fly.

## Start

```shell
SERVER_URI="my-server-base-uri" sbt run
```

or
```shell
sbt run -Dserver.uri="my-server-base-uri"
```

## Execute tests

```shell
sbt test
```


## Current Features

- transparent proxy
